# Infinite Navbar Component

Component files:
- `components/navbar/navbar.html`
- `components/navbar/navbar.css`
- `components/navbar/navbar.js`

Loader:
- `main.js` loads `navbar.html` into `#navbar-placeholder`
- `navbar.js` must be available on the page (included or dynamically injected)

Note: This repo currently includes a legacy `script.js`—navbar behavior is now implemented in `components/navbar/navbar.js`.
